## 👨‍💻 Markdown Cheat Sheet 👩‍💻

---
### 1️⃣ Headings
---

#### Heading 1
`# H1`

#### Heading 2
`## H2`

#### Heading 3
`### H3`

#### Heading 4
`#### H4`

#### Heading 5
`##### H5`

#### Heading 6
`###### H6`

---
### 2️⃣ Text Decoration
---

#### Bold
`**bold**`

#### Italic
`*italicized text*`

#### Line Through
`~~line through~~`

---
### 3️⃣ List
---

#### Ordered List
```
1. First Item
2. Second Item
3. Third Item
```

#### Unordered List
```
* First Item
* Second Item
* Third Item
```

---
### 4️⃣ Code Decoration
---

#### Code
`code`

#### Code Block

```html
<title>Hello World!</title>
<link rel="stylesheet" href="style.css">
```

---
### 5️⃣ Others
---

#### Checkbox
```
* [x] First Item
* [ ] Second item
```

#### Horizontal Rule
`---`

#### Link
`[Boostnote](https://boostnote.io/)`

#### Quote
`> This is a quote from somewhere!`
